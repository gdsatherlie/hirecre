# HireCRE

A Next.js + TypeScript + Tailwind prototype for a commercial real estate job board and content hub.

## Getting started

```bash
npm install
npm run dev
```

## Routes
- `/` – landing page with featured jobs, resources, and articles
- `/jobs` – job list with client-side filtering
- `/jobs/[id]` – job detail
- `/resources` – resources index
- `/resources/[id]` – resource detail
- `/blog` – blog index
- `/blog/[slug]` – blog detail
- `/dashboard/employer` – employer dashboard mock
- `/dashboard/candidate` – candidate dashboard mock
